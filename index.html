Feedback Assistant

A Chrome extension that helps you quickly insert pre-written feedback comments into Google Docs, Sheets, and Slides using a personalised comment bank stored in Google Sheets.

What Feedback Assistant Does

Feedback Assistant allows users to create and manage a bank of feedback comments in Google Sheets. Each comment is linked to a short abbreviation.

When working in Google Docs, Sheets, or Slides, users can type an abbreviation and the extension automatically replaces it with the full feedback comment.

How It Uses Google Data

The extension uses Google authentication to securely access the user’s own Google Drive and Google Sheets.

It only reads data from user-selected spreadsheets to retrieve abbreviations and associated feedback comments.

The extension does not access emails, contacts, or any unrelated data.

Permissions

Feedback Assistant requests access to Google Drive and Google Sheets solely to load and read the user’s selected comment bank.

It also interacts with Google Docs, Sheets, and Slides to insert feedback comments based on user input.

Privacy

Feedback Assistant does not collect personal data, track user activity, or transmit data to external servers.

All data remains within the user’s Google account or local browser storage.
